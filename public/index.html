Upload Data

Kirim file teks ke API (dari skrip lokal / curl):

curl -X POST --data-binary @userwdp.txt http://IP:8080/api/upload
curl -X POST --data-binary @hasil.txt http://IP:8080/api/upload-hasil
curl -X POST --data-binary @userlimit.txt http://IP:8080/api/upload-limit

Format File

userwdp.txt → user|id|jumlah

hasil.txt → user|id|link_invoice

userlimit.txt → user|id|lainnya
