WDP Sheet

API Node Express + tampilan spreadsheet untuk copy-paste ke Google Sheets.

Buka Sheet →

Upload via Controller

Gunakan menu PHP di mesin Anda (bukan upload browser):

php controller.php

Atau perintah cepat:

./upload-data userwdp.txt
./upload-hasil hasil.txt
./upload-limit userlimit.txt
./open-sheet

Format File

userwdp.txt → user|id|jumlah

hasil.txt → user|id|link_invoice

userlimit.txt → user|id|lainnya

Contoh ada di folder files/

Tampilkan Data

Urutan: sukses → limit → zonk. Kolom User merah = limit.
